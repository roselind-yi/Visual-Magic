# BoLiMusic - Music Visualization Project

## Project Introduction

BoLiMusic is a web-based music visualization project that can real-time analyze audio data and generate various dynamic visualization effects. This project supports multiple audio formats and provides an intuitive user interface, allowing users to experience the charm of music through visual means.

## Features

- **Audio File Upload**：Supports both drag-and-drop and browse methods for uploading audio files
- **Real-time Visualization**：Provides multiple visualization effects, including waveform, particles, geometry, raindrops, number 10, and keyboard effects
- **Audio Analysis**：Uses Web Audio API for real-time audio analysis
- **Responsive Design**：Adapts to different screen sizes
- **Audio Tag Parsing**：Automatically extracts song information and album covers

## Technology Stack

- **Frontend Framework**：Pure native JavaScript
- **UI Framework**：Tailwind CSS
- **Audio Processing**：Web Audio API
- **Visualization**：Canvas API
- **Audio Tag Parsing**：jsmediatags

## Quick Start

### Method 1: Direct Opening

1. Find the `BoLiMusic` folder
2. Double-click the `index.html` file to open it in a browser
3. Upload an audio file (you can use the example music in the `examples_music` folder)
4. Start experiencing music visualization effects

### Method 2: Using Local Server (Recommended)

1. Open a command line tool and enter the `BoLiMusic` folder
2. Start the local server:
   ```bash
   node server.js
   ```
3. Visit `http://localhost:8080` in your browser
4. Upload audio files and experience visualization effects

## How to Use

1. **Upload Audio Files**：
   - Click the "Upload Audio File" button
   - Or directly drag and drop audio files to the designated area

2. **Playback Control**：
   - Click the play button to start/pause music
   - Observe the progress bar and time display

3. **Switch Visualization Effects**：
   - Click the visualization style buttons at the bottom
   - Select different effect styles

4. **Experience Different Music**：
   - Try uploading music of different styles
   - Observe the changes in visualization effects
